# 🚀 ACV: Azure Critical Validator (애사비)

> **YouTube 콘텐츠의 신뢰도를 Azure AI로 검증하고 제어하는 크롬 익스텐션 솔루션**

## 📌 프로젝트 개요

**ACV(애사비)**는 유튜브 내의 무분별한 허위 정보와 신뢰도가 낮은 콘텐츠로부터 사용자를 보호하기 위해 개발되었습니다. 클라이언트 사이드에서 실시간으로 자막 데이터를 수집하고, Azure 기반의 AI 분석을 통해 영상의 신뢰도 점수를 산출하여 시각적인 제어(블러 처리 및 재생 정지)를 제공합니다.

## ✨ 주요 기능

* **실시간 신뢰도 검증** : 영상 진입 시 Azure AI와 Databricks를 통해 콘텐츠 신뢰도 점수 산출.
* **강력한 콘텐츠 제어** : 신뢰도가 낮은 영상(DANGER) 판정 시, 영상 블러 처리 및 물리적 재생 정지(광고 포함).
* **Toss 스타일 UI/UX** : 사용자 친화적인 미니멀리즘 디자인과 직관적인 인터랙션 제공.
* **인카드 상세 리포트** : 영상 시청 중에도 페이지 이동 없이 분석 근거를 바로 확인할 수 있는 토글 리포트.
* **사이드 정보 패널** : 안전한 영상 시청 시 방해되지 않는 우측 사이드 패널로 신뢰 점수 확인 가능.

## 🏗️ 시스템 아키텍처

기존 서버 사이드 수집의 한계를 극복하기 위해 **클라이언트 기반 수집 아키텍처**로 설계되었습니다.

1. **익스텐션** : 유튜브 페이지 내 HTML에서 자막(Captions) 데이터 직접 추출.
2. **서버 통신** : SSE(Server-Sent Events) 커넥션을 통해 추출된 데이터를 서버로 실시간 전송.
3. **데이터 분석** : Databricks(`acv_adb` 카탈로그) 내에서 LLM 전처리 및 신뢰도 분석 수행.
4. **결과 피드백** : 분석 완료 즉시 익스텐션 UI로 신뢰도 배지 및 경고창 렌더링.

## 🛠️ 기술 스택

* **Frontend** : JavaScript (Vanilla JS), CSS3 (Toss 스타일 가이드), HTML5
* **Backend** : FastAPI (Python), SSE (Server-Sent Events)
* **Infra/Data** : Azure, Databricks
* **Extension** : Manifest V3 기반

## 👥 팀원 역할

* **PM & UI/UX** : 익스텐션 UI 설계, 블러 로직 구현, 신뢰도 배지 규격 확정 및 AI 윤리 정책 수립.
* **Back-end & Logic** : 자막 추출 로직 구현, SSE 통신 서버 구축 및 FastAPI-Databricks 연동.
* **Infra & Data** : 데이터브릭스 인프라 구축, 카탈로그 및 스키마 권한 관리, 데이터 무결성 제어.
* **Data & Pre-processing** : 자막 데이터 정제 및 LLM 분석 최적화 파이프라인 개발.

## 🚀 설치 및 시작하기

1. 이 레포지토리를 로컬에 클론합니다.
2. 크롬 브라우저에서 `chrome://extensions` 페이지로 이동합니다.
3. 오른쪽 상단의 **'개발자 모드'**를 활성화합니다.
4. **'압축해제된 확장 프로그램을 로드합니다'** 버튼을 클릭하고 프로젝트 폴더를 선택합니다.
5. 유튜브 `watch` 페이지에서 ACV의 실시간 검증 기능을 확인하세요.
